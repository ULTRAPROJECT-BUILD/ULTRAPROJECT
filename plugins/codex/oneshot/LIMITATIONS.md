# Limitations

- This package is local/repo-ready. It does not claim submission to an official marketplace.
- Codex uses the bundled skill and starter prompts. This package does not promise Claude slash-command behavior or a portable literal `/ONESHOT` command.
- Optional app, MCP, and hook manifests are omitted because this package does not bundle a new app, MCP server, or hook.
- Credentialed integrations, spending, external communications, payments, and legal actions require operator setup, permission, and review.
- Long-running project execution still depends on the active Codex session and the local OneShot workspace. There is no hidden daemon.
- Live marketplace install checks may vary by Codex CLI version, local configuration, and authentication state.
- Historical provenance files in the broader repository may preserve source-project attribution where changing it would falsify history.
